I built a feature for my static website in another exercise in which a search icon was added to the top. In clicking this search icon, there would be an overlay in which you could search up books for this static website. Involved in this was adding an overlay div that darkened the rest of the screen behind the overlay.

Micro iteration felt definitely a lot different, however it also gave me a lot more control than having claude look into and doing everything for me. Overall, it was a weird but enjoyable experience.

The self-review actually caught a couple of things, such as styling errors with the text, as well as simple things like being able to have multiple modals open at the same time, creating conflicts.

I liked Claude Web a lot, however I found that I was sometimes limited by what Github would allow Claude to do compared to whenever I would use Claude Code and it could directly edit my local files.
